Funciones combinadas matemáticas

Combinando algunas de las funciones matemáticas resuelve los ejercicios.

Ejemplo

Funciones predefinidas matemáticas
Script que genere un número aleatorio entero con 101 valores posibles entre -50 y 50 y lo muestre por pantalla.

/* SQL Solución con MOD, FLOOR y RAND */
/* Esto genera un numero entre 0 y 1 (0 ≤ num < 1) */
SET @num = RAND();

/* Multiplico por un número arbitrario cualquiera que sea mayor que 101; por ejemplo 10000. Con esto tendremos que: 0 ≤ num < 10000 */
SET @num = 10000 * @num;

/* Me quedo con la parte entera. Por lo tanto el número num es tal que: 0 ≤ num < 10000 y num es entero */
SET @num = FLOOR(@num);

/* Con esto nos quedamos con el resto de la división entera de num entre 101 (fijaos que nos dicen que hay 101 valores posibles). Como el resto es mayor o igual que 0 y menor que el divisor (101) tenemos que ahora num es tal que: 0 ≤ num <= 100. Ya casi lo tenemos */
SET @num = MOD(@num, 101);

/* Este paso es muy fácil de entender; tenemos que el valor más pequeño posible de num es 0. Si sumamos -50 a num tendremos que en este caso el valor más pequeño será -50 (0 + (-50) = 0 – 50 = -50) y el más grande será 50 (100 + (-50) = 100 – 50 = 50). */
SET @num = @num - 50;

/* Por último lo visualizamos con: */
SELECT @num;

/* Se puede simplificar todo el proceso agrupando las funciones; es decir pasando el resultado de una función como parámetro de otra. La forma más simplificada tendría este aspecto: */
SET @num = MOD(FLOOR(10000 * RAND()), 101) - 50;
SELECT @num;

/* Cualquiera de las dos soluciones es correcta o se puede optar por una opción intermedia como la siguiente */
/* La primera instrucción almacena en @num un entero mayor o igual que 0 y menor que 10000 */
SET @num = FLOOR(10000 * RAND());

/* Esta instrucción calcula el resto de la división entre 101 (el resultado arrojará un valor entre 0 y 100) y le suma -50 (o resta 50) obteniendo lo que nos piden un valor entre -50 y 50.*/
SET @num = MOD(@num, 101) - 50;
SELECT @num;

SET @a = 100 * RAND();
SET @b = FLOOR(@a);
SELECT @b;

-- Añternativa en una sola línea
SET @a = FLOOR(100 * RAND());
SELECT @a;

✅ Ejercicio 5

Genera 3 campos alaeatoriamente correspondientes a la hora, minutos y segundos. Por ejemplo, como salida podríamos tener algo similar a esto:

+------+---------+----------+
| HORA | MINUTOS | SEGUNDOS |
+------+---------+----------+
|   10 |      25 |       34 |
+------+---------+----------+

SET @segundos = FLOOR(60 * RAND());

✅ Ejercicio 6

Crear una variable global denominada num con un valor aleatorio entre -10 y +10 y luego obtener el signo, el valor absoluto, y su potencia al cubo y mostrar esos valores.

💡 Solución

SET @num = FLOOR(21 * RAND()) + (-10);
SELECT @num AS NUMERO, SIGN(@num) AS SIGNO, ABS(@num) AS VALOR_ABSOLUTO, POW(@num, 3) AS POTENCIA_CUBO;

Funciones combinadas de cadenas o strings

Combinando algunas de las funciones cadenas resuelve los ejercicios.

Ejemplo 2

Funciones de cadenas combinadas
Dada una url obtener y mostrar el nombre del servidor y el del dominio. Por ejemplo si tenemos la url:

http://www.iesdoctorbalmis.com/ofertaeducativa/informatica/index.php

el resultado será:

+------------------------------+
| RESULTADO                    |
+------------------------------+
| SERVIDOR: www                |
| DOMINIO: iesdoctorbalmis.com |
+------------------------------+

SET @cad = 'http://www.iesdoctorbalmis.com/ofertaeducativa/informatica/index.php';

/* Esta instrucción obtiene la posición donde empieza la cadena '://' en la url dada. En nuestro ejemplo devolverá 5. */
SET @pos = INSTR(@cad, '://');

/* Ahora de la variable @cad obtememos la subcadena que empieza en la posición calculada anteriormente más 3 (se suma 3 por que '://' tiene 3 caracteres. En nuestro caso nos quedaríamos con la subcadena que empieza en la posición 8 de cadena hasta el final. Se sobreescribe el valos de la variable @cad con el nuevo valor. En nuestro ejemplo @cad valdría:
'www.iesdoctorbalmis.com/ofertaeducativa/informatica/index.php' */
SET @cad = SUBSTRING(@cad, @pos + 3);

/* En la variable @nombreDns guardamos la subcadena de @cad que está a la izquierda de la primera aparición del carácter '/'. En nuestro ejemplo @nombreDns valdría: 'www.iesdoctorbalmis.com' */
SET @nombreDns = SUBSTRING_INDEX(@cad, '/', 1);

/* Ahora obtenemos el nombre del servidor de la variable  @nombreDns coquedándonos con lo que hay a la izquierda de la primera aparición del carácter '.'. En nuestro ejemplo @servidor valdría: 'www'*/
SET @servidor = SUBSTRING_INDEX(@nombreDns, '.', 1);

/* Ahora obtenemos la posición donde está el primer carácter '.' en la cadena @nombreDns */
SET @pos = INSTR(@cad, '.');

/* Obtenemos el dominio de @nombreDns cogiendo el substring que empieza en la posición siguiente a @pos y va hasta el final. En nuestro ejemplo @dominio valdría: 'iesdoctorbalmis.com' */
SET @dominio = SUBSTRING(@nombreDns, @pos + 1);

/* Por último utilizo la función CONCAT para que se muestren los resultados tal como nos los piden. */
SELECT CONCAT('SERVIDOR: ', @servidor, '\nDOMINIO: ', @dominio) AS RESULTADO;

✅ Ejercicio 7

Extraer de una cadena la primera palabra. Utilizad las funciones INSTR y LEFT.

/* Procedimiento a seguir:
    1. Asignar a una variable la cadena que nos dan
    2. Buscar la posición del primer espacio en blanco
    3. Obtener la cadena desde el inicio hasta el espacio en blanco
*/

💡 Solución

SET @p = 'Extraer primera palabra';
SET @pos = INSTR(@p, ' ');
SET @res = LEFT(@p, @pos - 1);
SELECT @res;

✅ Ejercicio 8

De una página web extraer su parte principal de dominio. Por ejemplo de http://iesdoctorbalmis.edu.gva.es/j16/index.php/es/ofertaeducativa obtenemos iesdoctorbalmis.edu.gva.es.
Utilizad las funciones REPLACE y SUBSTRING_INDEX.

/* Procedimiento a seguir:
    1. Asignar a una variable la cadena que nos dan
    2. Eliminar http://
    3. Obtener la parte inicial de la cadena hasta la primera barra
*/

💡 Solución

SET @p = 'http://iesdoctorbalmis.edu.gva.es/j16/index.php/es/ofertaeducativa';
SET @cad = REPLACE(@p, 'http://', '');
SET @res = SUBSTRING_INDEX(@cad, '/', 1);
SELECT @res;

Funciones combinadas de fechas

Combinando algunas de las funciones de fechas resuelve los ejercicios.

Ejemplo 3

Funciones de fechas combinadas
Mostrar el día de la semana del día 25 del mes en el que estamos (en el idioma por defecto; normalmente en inglés).

/* Obtengo la fecha actual (CURDATE) */
SET @fecha = CURDATE();

/* Obtengo el mes correspondiente a la fecha actual */
SET @mes = MONTH(@fecha);

/* Obtengo el año correspondiente a la fecha actual */
SET @year = YEAR(@fecha);

/* Construyo una cadena con formato dd/mm/yyyy con el día 25 del més actual y el año actual */
SET @cadenaNuevaFecha = CONCAT(25,'/',@mes,'/',@year);

/* Paso la cadena de caracteres con la fecha a una variable de tipo DATE */
SET @nuevaFecha = STR_TO_DATE(@cadenaNuevaFecha,'%d/%m/%Y');

/* Obtengo el nombre del día de la semana (en inglés) */
SET @diaSemana = DAYNAME(@nuevaFecha);

/* Muestro el día de la semana */
SELECT @diaSemana;

Ejemplo 3bis

Funciones de fechas combinadas
Mostrar el día de la semana del día 25 del mes en el que estamos en español.

/* Obtengo la fecha actual (CURDATE) */
SET @fecha = CURDATE();

/* Obtengo el mes correspondiente a la fecha actual */
SET @mes = MONTH(@fecha);

/* Obtengo el año correspondiente a la fecha actual */
SET @year = YEAR(@fecha);

/* Construyo una cadena con formato dd/mm/yyyy con el día 25 del més actual y el año actual */
SET @cadenaNuevaFecha = CONCAT(25,'/',@mes,'/',@year);

/* Paso la cadena de caracteres con la fecha a una variable de tipo DATE */
SET @nuevaFecha = STR_TO_DATE(@cadenaNuevaFecha,'%d/%m/%Y');
   
/* Hasta aquí todo igual que el caso anterior */

/* Obtengo el número del día de la semana. Hay que tener en cuenta que el primer día es el domingo. */
SET @numDiaSemana = DAYOFWEEK(@nuevaFecha);

/* En función del valor del número de día de la semana muestra el nombre correspondiente en español. */
SELECT CASE
           WHEN (@numDiaSemana = 1) THEN 'Domingo'
           WHEN (@numDiaSemana = 2) THEN 'Lunes'
           WHEN (@numDiaSemana = 3) THEN 'Martes'
           WHEN (@numDiaSemana = 4) THEN 'Miércoles'
           WHEN (@numDiaSemana = 5) THEN 'Jueves'
           WHEN (@numDiaSemana = 6) THEN 'Viernes'
           WHEN (@numDiaSemana = 7) THEN 'Sábado'
           ELSE 'Error en día'
       END
AS DIA_SEMANA;

✅ Ejercicio 10

Calcular la fecha del próximo domingo desde la fecha actual. Se pide una solución utilizando CURDATE, DAYOFWEEK y ADDDATE.

/* Procedimiento a seguir:
    1. Asignar a una variable la fecha de hoy
    2. Calcular días que faltan hasta el próximo domingo
    3. Calcular fecha hasta el próximo domingo
*/

💡 Solución

SET @hoy = CURDATE();
SET @diaSemana = DAYOFWEEK(@hoy);
SET @diasRestantes = (7 - @diaSemana) + 1;
SET @nuevaFecha = ADDDATE(@hoy, @diasRestantes);
SELECT @nuevaFecha;

✅ Ejercicio 11

Dada una fecha en formato dd/mm/yyyy obtener la fecha en formato: nombreMes dia, año que es 50 días anterior a la dada (nombreMes está en inglés). Son necesarias las funciones ADDDATE, STR_TO_DATE y DATE_FORMAT. Por ejemplo para la fecha 17/03/1965 obtendríamos January 26, 1965.

💡 Solución

SET @cadFecha = '17/03/1965';
SET @fecha = STR_TO_DATE(@cadFecha, '%d/%m/%Y');
SET @nuevaFecha = ADDDATE(@fecha, -50);
SET @cadNuevaFecha = DATE_FORMAT(@nuevaFecha, '%M %d, %Y');
SELECT @cadNuevaFecha;

✅ Ejercicio 12

Partiendo de una fecha, indicar si es hoy, pasado o futuro. Se pide una solución utilizando CURDATE y DATEDIFF.

/* Procedimiento a seguir:
    1. Almacenar la fecha en variable (lo que nos dan)
    2. Comparar valor de fecha con CURDATE
*/

💡 Solución

SET @fecha = '1965-03-17';
SET @dias = DATEDIFF(CURDATE(), @fecha);
SELECT CASE
   WHEN  @dias > 0 THEN 'PASADO'
   WHEN  @dias < 0 THEN 'FUTURO'
   ELSE 'HOY'
END;
